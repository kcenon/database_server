Detailed Description

CRTP-based collector for database query metrics.

Implements the query_collector_base interface for comprehensive metrics collection. All operations are thread-safe using atomic operations and minimal locking.

Thread Safety:

  • All collection methods are thread-safe
  • Statistics retrieval uses shared locks
  • Metrics storage uses atomic operations
