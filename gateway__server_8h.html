Detailed Description

Database gateway TCP server.

Implements the network layer for the database gateway using kcenon::network. Handles client connections, message routing, and lifecycle management.

Architecture:

  • Uses tcp_facade / i_protocol_server from kcenon::network for TCP handling
  • Integrates with query_protocol for message serialization
  • Provides callbacks for request handling
