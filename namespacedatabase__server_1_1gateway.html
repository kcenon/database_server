◆ generate_session_id()

std::string database_server::gateway::generate_session_id ( )

Generate a cryptographically secure session ID.

Creates a 32-character hexadecimal string representing 128 bits of pseudo-random data seeded from hardware entropy.

Thread Safety:

  • This function is thread-safe
  • Uses thread-local RNG for optimal performance

Performance:

  • Expected generation time: < 1s
  • No locking required between threads
Returns
A 32-character lowercase hexadecimal string

Example:

auto session_id = generate_session_id();
// session_id == "a1b2c3d4e5f6789012345678abcdef01"
