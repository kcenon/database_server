Detailed Description

template<typename Handler>
class database_server::gateway::query_handler_wrapper< Handler >

Type erasure wrapper for CRTP handlers.

Wraps a CRTP-based handler to provide a virtual interface for runtime polymorphism. The actual handler operations use CRTP for zero-overhead dispatch.

Template Parameters
HandlerThe CRTP handler type to wrap

Usage Example:

auto handler = std::make_unique<query_handler_wrapper<select_handler>>();
registry.register_handler(std::move(handler));
