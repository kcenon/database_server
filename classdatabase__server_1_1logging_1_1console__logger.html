Detailed Description

Simple console logger implementing ILogger interface.

This logger outputs messages to stdout (info and below) or stderr (warning and above). It provides thread-safe logging with configurable log levels and formatted output.

Thread Safety:

  • All logging methods are thread-safe
  • Level changes are atomic

Usage:

auto logger = database_server::logging::create_console_logger();
logger->log(kcenon::common::interfaces::log_level::info, "Server started");
