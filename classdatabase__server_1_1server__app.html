Detailed Description

Main database server application.

This class manages the complete lifecycle of the database gateway server. It handles configuration loading, server initialization, running the main event loop, and graceful shutdown.

Thread Safety:

  • The start() and stop() methods are thread-safe
  • State transitions are atomic

Usage Example:

database_server::server_app app;
if (!app.initialize("config.yaml")) {
return 1;
}
return app.run();
