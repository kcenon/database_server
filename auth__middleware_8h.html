Detailed Description

Authentication middleware for database gateway.

Provides authentication validation, rate limiting, and audit logging for the database gateway server. Designed to be pluggable with custom token validators.

Features:

  • Token-based authentication with pluggable validators
  • Per-client rate limiting with sliding window
  • Audit logging for security events
  • Thread-safe metrics collection
