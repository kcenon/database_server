Detailed Description

Console logger implementation for database_server.

Provides a simple ILogger implementation that outputs to stdout/stderr. This serves as the foundation for unified logging infrastructure.

Features:

  • Implements kcenon::common::interfaces::ILogger
  • Thread-safe console output
  • Configurable log levels
  • Timestamps and log level prefixes
