Detailed Description

template<typename Derived>
class database_server::metrics::query_collector_base< Derived >

CRTP base class for query metrics collectors.

This template class implements common functionality shared by all query metrics collectors:

  • Configuration parsing (enabled state)
  • Collection with error handling and statistics
  • Health monitoring
  • Statistics tracking (collection count, error count)
